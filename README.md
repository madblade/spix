# spix

### Preview

1. Deploy.
2. Click on 'Request'.
3. Middle-click to request pointerLock (either Firefox or Chrome will do).
4. Mouse+WASD, R/L-click, F, G, R, ESC to show settings.

### Deploy locally

`$> git clone https://github.com/madblade/spix`

`$> cd ${spix}`

`$> npm install`

`$> bower install`

`$> grunt serve`
